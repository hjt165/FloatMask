# FloatMask 项目总结

## 一、项目概述

FloatMask是一款基于Android平台的悬浮窗工具应用，旨在帮助外语学习者通过遮挡字幕来提升听力能力。用户可以自由拖动和调整悬浮框大小，将其覆盖在视频字幕区域，实现无字幕听力训练。

**技术栈**：Python + Kivy + Buildozer

**目标平台**：Android 7.0+ (API 24+)

---

## 二、已实现功能

### 2.1 P0阶段：基础框架搭建 ✅
- 项目目录结构创建
- Kivy环境配置
- 权限检测模块（Android悬浮窗权限）
- 悬浮框基础显示（200×120像素，半透明灰色）
- 基础拖动功能（单指拖动，实时更新）

### 2.2 P1阶段：核心功能完善 ✅
- 尺寸调整功能（四角和四边控制点）
- 边界限制（至少20%在屏幕内）
- 最小/最大尺寸限制（80×80到30%×30%）
- 后台服务（OverlayService + KeepAliveService）
- 退出清理（atexit + Window.on_close + App.on_stop）

### 2.3 P3阶段：增强交互功能 ✅
- 双击颜色切换（灰色/黑色/纯黑循环）
- 长按快捷菜单（颜色、透明度、最小化、关闭）
- 记忆功能（SharedPreferences/JSON持久化）
- 最小化模式（点击箭头缩成小圆点）

## 四、技术架构

```
FloatMask/
├── app/                    # 主应用目录
│   ├── main.py            # 应用入口
│   ├── ui/                # 用户界面
│   │   ├── overlay_widget.py    # 悬浮框组件
│   │   └── quick_menu.py       # 快捷菜单
│   ├── core/              # 核心功能
│   │   ├── permission_handler.py  # 权限处理
│   │   ├── boundary_calculator.py # 边界计算
│   │   └── gesture_handler.py    # 手势处理
│   ├── services/          # 后台服务
│   │   └── overlay_service.py    # 悬浮窗服务
│   └── utils/             # 工具函数
│       └── config.py      # 配置管理
├── resources/             # 资源文件
├── tests/                 # 测试文件
├── docs/                  # 文档
├── buildozer.spec         # Android打包配置
└── requirements.txt       # Python依赖
```
